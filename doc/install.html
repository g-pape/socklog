socklog - installation

check that you have a recent version of daemontools installed.

unpack socklog_x.x.x.tar.gz 

  # tar xfzvp socklog_x.x.x.tar.gz
  # cd socklog-x.x.x
on solaris, edit the top of Makefile, add -DSOLARIS to CFLAGS and -lsocket -lnsl -ldoor -lthread to LDFLAGS.

run make and make install 

  # make
  # make install
refer to Configuration for configuring and setting up socklock services.
