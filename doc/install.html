socklog - installation

since version 0.3.0, socklog installs into /package.

check that you have the recent version of daemontools installed.

create a /package directory 

  # mkdir -p /package
  # chmod 1775 /package
  # cd /package
download socklog-0.3.0.tar.gz into /package and unpack the archive 
  # gunzip socklog-0.3.0.tar
  # tar -xpf socklog-0.3.0.tar
  # rm socklog-0.3.0.tar
  # cd admin/socklog-0.3.0
on solaris, edit the top of src/Makefile, add -DSOLARIS to CFLAGS and -lsocket -lnsl -ldoor -lthread to LDFLAGS.

compile and install the socklog program 

  # package/install
gzip the man page and copy it into a reasonable directory in your $MANPATH, e.g. 
  # mkdir -p /usr/local/man/man8/
  # gzip doc/socklog.8 && cp -f doc/socklog.8.gz /usr/local/man/man8/
refer to Configuration for configuring and setting up socklock services.
