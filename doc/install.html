socklog - installation

Since version 0.3.0, socklog installs into /package.

Check that you have the recent version of daemontools installed.

Create a /package directory 

  # mkdir -p /package
  # chmod 1775 /package
  # cd /package
Download socklog-0.5.3.tar.gz into /package and unpack the archive 
  # gunzip socklog-0.5.3.tar
  # tar -xpf socklog-0.5.3.tar
  # rm socklog-0.5.3.tar
  # cd admin/socklog-0.5.3
On solaris, edit the top of src/Makefile, add -DSOLARIS to CFLAGS and -lsocket -lnsl -ldoor -lthread to LDFLAGS.

Compile and install the socklog programs 

  # package/install
gzip the man pages and copy them into a reasonable directory in your $MANPATH, e.g. 
  # mkdir -p /usr/local/man/man8/
  # gzip man/socklog.8 && cp -f man/socklog.8.gz /usr/local/man/man8/
  # mkdir -p /usr/local/man/man1/
  # gzip man/tryto.1 && cp -f man/tryto.1.gz /usr/local/man/man1/ 
  # gzip man/uncat.1 && cp -f man/uncat.1.gz /usr/local/man/man1/
Refer to Configuration for configuring and setting up socklog services.
