socklog - upgrading from previous versions

0.4.0 or 0.4.1 to 0.4.3

The named pipe for log events notification has now more strict permissions for security reasons. It is recommended to upgrade this service if it is running by recreating the service directory following the instructions.

The socklog program and the uncat program are now run by softlimit.
